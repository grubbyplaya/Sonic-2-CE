A currently unfinished port of the Sega Master System version of Sonic 2 to the TI-84+ CE.

So far, the game assembles into a 36KB .8xp file, but launching it will crash your calculator. 

# Controls

Arrow keys - Move Sonic

2nd/Alpha - Jump
